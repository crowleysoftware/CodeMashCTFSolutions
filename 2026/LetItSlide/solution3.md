# Let it Slide

## Challenge

Given TWYIDSLVPSFPAUVMQFHWF and told CODEMASH was key to deciphering.

## Solution

Use dcode's [Vigenere cipher decoder](https://www.dcode.fr/vigenere-cipher):

- Ciphertext: TWYIDSLVPSFPAUVMQFHWF
- Knowing the Key: CODEMASH

-> Decrypt

Returns the flag: `CTF{RIVER-STONE-CLOUD-FOREST}`
